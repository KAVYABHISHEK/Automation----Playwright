📦 Batch — top-level Playwright suite folder
📁 Module — feature / page grouping

ℹ️
How this maps to your Playwright project: Each Batch is a top-level suite folder (e.g. tests/<batch>/), and every Module inside it holds two sub-folders — testcases/ for your .spec.ts files, and baselines/ for the reference screenshots that Playwright's toHaveScreenshot() compares against. Saving a test case here scaffolds it into testcases/ under the Batch/Module you choose. The first Quick Run for that test case captures and stores its baseline at baselines/<jsonId>.png; every run after that compares the live capture against it.

⚠ Frontend prototype only: nothing here is sent to a server or saved to disk/browser storage — refreshing the page resets the demo data shown below.
